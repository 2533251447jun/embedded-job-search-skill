# Application Message Template

All messages must be manually reviewed and sent by the user. Do not auto-apply, auto-DM, auto-send, auto-contact HR, or bypass platform rules.

## 安全使用字段

- 适用岗位族：STM32/MCU 固件 / 嵌入式测试 / 硬件调试 / IoT / 工业控制/RS485/Modbus / BSP/Linux 初级 / 车载测试/CAN/UDS
- 适用平台：Boss 直聘 / 智联招聘 / 前程无忧 / 猎聘 / 邮箱 / 内推
- 消息类型：首次沟通 / HR 回复 / 补充资料 / 感谢信 / 面试后跟进 / 二次联系 / 简历补充说明 / 项目补充说明
- 可引用证据：简历 / Git / 课程设计 / 项目报告 / 作品集 / 串口日志 / 测试报告 / 调试记录 / 波形截图
- 不可引用内容：虚假企业经历 / 虚假量产经验 / 虚假客户 / 保密项目 / 不存在的代码仓库 / 不存在的测试报告 / 不存在的性能指标 / 不存在的团队规模
- 是否存在夸大风险：YES / NO / needs_proof
- 是否符合平台规则：YES / NO / needs_proof

## 邮件投递

```text
主题：【岗位名称】投递 - 【姓名/称呼】 - 【嵌入式方向】

您好，

我想投递贵司【岗位名称】。我的经历主要集中在【岗位族】方向，项目中涉及【芯片/外设/协议/测试/调试工具】。

附件为我的简历。简历中的项目均按真实经历整理；若涉及课程/个人项目，我已在项目说明中保留边界。期待有机会进一步沟通。

谢谢。
```


## 面试后跟进

```text
您好，感谢今天的面试。面试中提到的【技术点】我会继续补充整理。关于【项目边界或短板】，我会按真实经历说明，不做夸大。期待后续反馈。
```

## 简历补充说明

```text

```

