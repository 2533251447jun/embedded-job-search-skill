## 岗位族专项路由

| 岗位族 | 重点检查 | 不得混写 |
|---|---|---|
| STM32 / MCU 固件 | 外设、C、HAL/LL、日志、基础调试 | 不写成 BSP/Linux 或架构设计 |
| 嵌入式测试 | 用例、日志、缺陷复现、回归、报告 | 不写成固件开发负责人 |
| 硬件调试 / 联调 | 供电、接线、波形、I2C/SPI/UART 问题 | 不写成硬件设计 |
| IoT 设备端 | MQTT、联网、心跳、断线重连、设备日志 | 不写成云平台负责人 |
| 工业控制 / RS485 / Modbus | CRC、轮询、超时、寄存器、异常码 | 不写成完整协议栈设计 |
| BSP / Linux 初级 | Device Tree、dmesg、sysfs、交叉编译 | 不写成驱动专家 |
| 车载测试 / CAN / UDS | CAN、UDS、DBC、测试用例、工具证据 | 无 CANoe/HIL 证据不写平台经验 |
